Tutorial

Clicando no ícone representado por uma seta, o usuário retornará para a página anterior.

Clicando na imagem, essa será ampliada para melhor visualização do conteúdo.

Clicando em “i”, o usuário é direcionado para a página de tutorial.

Clicando no ícone representado por uma casa, o usuário volta para a tela principal.

Clicando no ícone representado por três linhas, o usuário será apresentado ao menu.

Clicando em “?”, o usuário é direcionado para a página de perguntas.

Clicando no ícone representado por um quebra-cabeça, o usuário é direcionado para a página de desafios.
